### 2 第一个C++程序
所以为什么是Hello Word?

这是一个非常经典的程序，能够帮助我们快速认识一个C++程序是怎样的
```C++
#include <iostream>

int main() {
    std::cout << "Hello World!" << std::endl;
    return 0;
}
```
非常简单，对吧，接下来我们逐行拆解这段程序：

```C++
#include <iostream>
```
这句代码的作用是告诉IDE我们引入了一个叫做`iostream`的头文件。那么，这个时候好奇的你又要问，什么是头文件？打个比方，如果你要写字，就要有笔这个工具。对于一个C++程序而言，头文件就像是程序做各种事情的、提前准备好的“工具”。你当然可以把这些工具的实现放进一个.cpp文件，但那样将会非常、非常的乱。我想大多数正常人都不会这样干！<br/>
当然，C++提供了各种各样的头文件，它们实现了各种不同的功能，你也可以使用其他第三方的头文件，甚至自己写一个头文件（后面会讲到的）！

```C++
int main() {
```
接下来我有必要向你介绍函数的概念——这可能听起来比较抽象，但是我们不会在这里做太多展开。现在，你只需要知道长这个样子的是一个函数，`main`是这个函数的名字。每个完整的C++程序都要有一个`main`函数，用作程序的入口。程序会从`main`函数内部的第一行代码（就是花括号框住的部分）开始执行<br/>
就现阶段而言，你只需要知道，写代码的时候，记得将你的代码框在`int main() {}`的**花括号**里面！

```C++
std::cout << "Hello World!" << std::endl;
```
这句代码就是整个程序的核心，它会在控制台打印一句`Hello World!`和一个换行符`endl`，而`std::out`就是我们头文件里的输出工具<br/>
请注意，打印的字符串要用双引包围起来，告诉程序这是一个字符串<br/>
另外要注意的一点是，所有的语句都要以分号结尾，否则该语句会被认为是不完整的语句

```C++
 return 0;
```
同样不会在这里作过多介绍，你只需要知道这个是函数的返回值，在这里的作用是宣告程序已经结束
