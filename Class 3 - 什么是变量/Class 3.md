### 3 多种多样的变量类型
还记得我们之前提到的这行代码吗？
```C++
int main() {
```
在上一节我们提到，这行代码声明了一个名为`main`的函数，而他前面的`int`代表了返回值的类型。在这里我仍然不会为你过多展开什么是函数的返回值——我们今天的重点是来认识像`int`这样的东西在C++里面到底是怎么一回事

请看下面的代码:
```C++
#include <iostream>

int main() {
    int a = 10;
    float b = 3.14;
    double c = 3.14;
    char d = 'A';
    long long e = 202604300000;

    std::cout << a << std::endl;
    std::cout << b << std::endl;
    std::cout << c << std::endl;
    std::cout << d << std::endl;
    std::cout << e << std::endl;

    return 0;
}
```
在这里，我将带领你们认识几个最基本的数据类型。但是在此之前，我要先告诉你们什么是变量：正如你所见，上面的代码块列出了一些看起来像是“等式”的东西。我们先取出一句来仔细拆解一下：
```C++
int a = 10;
```
这句代码组成如下：
* `int`：宣告了变量的数据类型，关于其他的变量的数据类型我们会在稍后介绍，这里`int`代表C++程序最基本的一种数据类型——整型
* `a`：这是这个变量的名字。变量的名字仅能由大小写字母、数字和下划线组成，且要遵循一定规则：
  * 变量名不能以数字开头
  * 变量名区分大小写
  * 不要将保留字作为变量名，这些保留字为C++内置的数据类型或有其他特殊含义。具体可参见 https://www.runoob.com/w3cnote/cpp-keyword-intro.html 来了解都有哪些保留字
* `=`：这是等于号吗？是，也不是。这个符号在C++的名字叫做赋值运算符，其作用是将等号右面的结果传递给等号左面的变量
* `10`：无需多言，这就是变量的值，一个数字10

可以说，变量就是你在程序中用来进行一系列操作的基础。大部分的内容都可以成为变量或者变量的抽象：数字、字符、文件、指针、甚至函数\

在C++里面，变量有以下几种类型：
- 基础类型（括号内代表其关键词/关键类型）：
    - 无类型（void）
    - (*自 C++ 11 起*) 空指针（std::nullptr_t)
    - 算术类型，包括：
        - 整数类型（int）
        - 浮点数类型（float，double）
        - 布尔类型（bool）
        - 字符类型（char）
- 复合类型

接下来介绍各个算术类型，其他类型在后续需要时再作介绍：

**1.布尔类型** \
布尔类型只能取两个值：`true`和`false` \
通常情况下，一个`bool`类型仅占用1字节空间

**2.整数类型** \
大多数情况下，最常用的整数类型是`int`，它能够表示从 -2,147,483,648 到 2,147,483,647 的整数，如果需要更大的数字，可以使用`long long`承载，它的取值范围要比`int`大得多！（从 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807）

**3.字符类型** \
该类型`char`用于承载单个字符（比如`a`这样的单个字母，或者是别的符号）。要知道的一点是，在C++，字符存储的底层方式仍是整数，通过 ASCII编码 使字符和整数一一对应。因此，我们可以通过数学运算来改变字符的内容 \
比如，`a + 1`在C++中可以被视为`b`

**4.浮点类型** \
主要包括两种：`float`和`double`。由于`float`类型表示范围较小（6-7 位有效数字），且精度不高，实际应用中常使用`double`（15 位有效数字）类型表示浮点数．
